      Para utilizar correctamente el ScrabbleAR, ejecutar el archivo "ScrabbleAR.py". 
   Se solicitará seleccionar la dificultad de juego y tras presionar el botón de 
   "Juego nuevo" dará inicio una partida de prueba.
   
      Para formar una palabra deberá seleccionar una letra del atril haciendo click
   sobre ella y luego presionar sobre un casillero del tablero. Las palabras se deben
   formar en sentido vertical u horizontal. Si se comete un error al formar la 
   palabra, podra seleccionar la opción "Modificar" para devolver las letras al atril.

      Tras haber construido la palabra, deberá presionar el botón "Evaluar" para 
   corroborar su validez y adjudicarse el puntaje correspondiente. En caso de no ser
   correcta, las fichas se reubicarán en el atril. Tras verificar la palabra, se 
   debe presionar "Pasar" para terminar el turno y que comienze el de la computadora.
